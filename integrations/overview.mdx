## Integration Categories

### AI Frameworks & RAG

Docling provides native integrations with popular AI frameworks:

- **LangChain** - Official document loader with full Docling support
- **LlamaIndex** - Reader and Node Parser components
- **Haystack** - Document converter for RAG pipelines
- **txtai** - Text extraction backend
- **Semantica** - Knowledge graph and semantic layer construction

### Agentic AI & Multi-Agent Systems

- **Crew AI** - Knowledge source for multi-agent workflows
- **MCP Server** - Standard protocol for AI agent tool integration
- **Pydantic AI** - Type-safe AI agent development
- **smolagents** - Lightweight agent framework

### Low-Code & Visual Platforms

- **Langflow** - Visual low-code platform with Docling components
- **Kotaemon** - Document reader for RAG applications
- **OpenWebUI** - User interface integration

### Data Processing & Annotation

- **Data Prep Kit** - Large-scale data preparation toolkit
- **Prodigy** - Document annotation and labeling
- **spaCy Layout** - NLP pipeline integration

### Cloud & Enterprise

- **Apify** - Cloud-based document processing
- **NVIDIA** - AI blueprints and enterprise solutions
- **Red Hat AI** - Enterprise AI platform integration

## Choose Your Integration

Select an integration based on your use case:

- **Building RAG applications?** Start with [LangChain](/integrations/langchain), [LlamaIndex](/integrations/llamaindex), or [Haystack](/integrations/haystack)
- **Creating AI agents?** Check out [MCP Server](/integrations/mcp-server) or [Crew AI](/integrations/crewai)
- **Working with knowledge graphs?** Explore [Semantica](/integrations/more#semantica)
- **Need cloud processing?** See [Apify](/integrations/more#apify)
- **Want visual development?** Try [Langflow](/integrations/more#langflow)
